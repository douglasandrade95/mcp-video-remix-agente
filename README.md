# MCP Video Remix Agente

Um MCP Server que integra Claude + Freepik + FFmpeg para remixagem automática de vídeos.

## Features

- 📹 Análise cinematográfica com Claude
- 🖼️ Geração de imagens com Freepik
- 🎬 Geração de vídeos com Freepik
- 🎵 Montagem automática com FFmpeg
- 🤖 Totalmente automático!
